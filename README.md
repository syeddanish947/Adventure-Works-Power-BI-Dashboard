Adventure Works Power BI Dashboard
A fully interactive business intelligence report built in Power BI using Adventure Works sales and returns data. This project covers everything from raw data modelling to advanced calculations and a polished, app-like report experience.

What is this project?
Think of this as a mini business dashboard — the kind companies use to monitor sales, track profits, understand customers, and make smarter decisions. Instead of scrolling through spreadsheets, everything is visual, interactive, and easy to explore.

Key Features
Clean Data Model
Data is organised using a star schema — a standard industry approach where one central table (sales or returns) connects to supporting tables like customers, products, calendar, and territory. This keeps everything fast and accurate.
20+ Custom DAX Measures
All the numbers you see — revenue, profit, return rate, monthly comparisons, targets, and trends — are powered by custom formulas written in DAX (Power BI's calculation language).
Drill-Through Navigation
Click on any product from the Executive Dashboard and drill straight through to its dedicated detail page. The entire page updates based on your selection.
Collapsible Filter Panel
A hidden slicer panel slides open when needed, keeping the dashboard clean by default. Built using bookmarks and button actions.
Hover Tooltips
Hovering over a category bar on the main dashboard pops up a live tooltip showing revenue, profit, orders, and return rate — without leaving the page.
What-If Price Adjustment
A slider on the Product Detail page lets you simulate how a price change would affect profit in real time.
AI Visuals
Two AI-powered pages use Power BI's built-in intelligence to surface patterns automatically — no manual filtering needed.

Tools and Skills Used
Power BI Desktop, DAX, Power Query, Star Schema Data Modelling, Bookmarks and Button Actions, Drill-Through, Report Page Tooltips, What-If Parameters, AI Visuals (Decomposition Tree and Key Influencers)

How to Open This Project

Download and install Power BI Desktop — it is free
Download the .pbix file from this repository
Open it in Power BI Desktop
Explore all 8 pages using the left-side navigation buttons inside the report


Dataset
Adventure Works is a sample dataset provided by Microsoft, commonly used for learning and practice. It contains fictional sales transactions, product details, customer records, and return data.

About This Project
This was built as a hands-on learning project to develop real-world Power BI skills including data modelling, DAX development, and interactive report design.
